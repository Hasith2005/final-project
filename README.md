To run this project, ensure you are connected to a GPU instance and have an LLM inference framework that supports the openai API.
Ensure that you also have the datasets. I have preprocessed the Feynman datasets into csv files. These have not been included due to filesize limits.
You can then create a venv and run the cells from feynman.ipynb.
The output will be plots showing the complexity-accuracy trade-offs between the guided and unguided versions.
